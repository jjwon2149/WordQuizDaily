## Documentation Rules

- 현재 구현과 목표 구현을 구분해 작성합니다.
- 서버 연동, 원격 DB, CMS, 계정 기반 학습 기록은 외국인 학습자 MVP 범위에 포함하지 않습니다.
- README와 문서 인덱스의 링크는 실제 존재하는 파일만 가리키도록 유지합니다.
- 채용/협업 리뷰어가 빠르게 판단할 수 있도록 "무엇을 만들었는지", "왜 그렇게 나눴는지", "어떻게 확인할 수 있는지"를 우선합니다.
