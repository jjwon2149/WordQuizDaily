## 문서 작성 기준

- MVP 문서는 현재 구현과 목표 구현을 구분해 작성합니다.
- 서버 연동, 원격 DB, CMS, 계정 기반 학습 기록은 외국인 학습자 MVP 범위에 포함하지 않습니다.
- README에서 연결하는 링크는 실제 존재하는 파일만 가리키도록 유지합니다.
